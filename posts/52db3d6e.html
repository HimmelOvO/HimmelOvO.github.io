CS50 01 Scratch
开始介绍了课程大概涵盖哪些范围，随后从二进制开始讲起，提到了一个byte是8位，所表示的范围256，所表达的每个数字与其他字符的对应就是ASCII码。
后来对其他语言和emoji之类的字符的需要，就增加位数，两个byte作为一个字，增加到三个、四个(计算机的8位、16位、32位)，而这种对应就是Unicode。
Unicode这个组织的使命是”to capture digitally all forms of human language”
emoji are not images, are characters defined in Unicode. 在不同的设备上不同是因为字体不同。
emoji的组合: one set of bits for the character on the left, one set of bits are for a character on the right, one set of bits for whatever emoji you want in the middle.
图片是一组像素，每个像素点都是一个数字；视频是一组变化的图片；音频是一组频率。
以IOS通讯录搜索名字为例，说明了分而治之的过程及伪代码逻辑展示。
